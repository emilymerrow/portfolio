MY JOURNEY IN CODING: PROJECT HIGHLIGHTS

Tomagotchi

Who said coding can't be fun? With this entertaining and interactive virtual pet simulator, I merged my love for programming with game development. Users can raise their own Tomagotchi by feeding, playing, and taking care of it – just like a real pet! But beware, neglecting your pet's needs can have dire consequences. Dive into this charming pet simulation and explore the joy of virtual pet ownership. This project was crafted using JavaScript, HTML, and CSS.

Children's Books Library Application

Welcome to the magical world of reading with the "Children's Books Library Application". This web-based application caters to young readers, offering a vast collection of children's books. Users can browse, add their favorite titles to a personalized list, and even contribute to the library by adding new books. The app is designed with user-friendly features like Google OAuth for secure authentication and a responsive design optimized for desktop viewing. Developed with Node.js, Express, and MongoDB, it offers full-stack CRUD functionality. The beauty of this application is its simplicity and convenience, creating a seamless user experience for book-loving children and their parents.

DEVentful

Introducing "DEVentful," an innovative event planning web application designed to simplify organizing events of any scale or type, from intimate birthdays to grand weddings and corporate events. Developed using Python, Django, PSQL, HTML, CSS, and JavaScript, this platform allows users to manage events, handpick vendors, review services, and maintain budgets with ease. To enhance user experience, we integrated Google Maps API for location services and enabled secure user authentication. The vision of DEVentful is to make event planning stress-free, giving you more time to revel in your special moments.

Chores Buster

Looking for a fun way to manage household tasks? Meet "Chores Buster," an inventive web app born out of a 9-year-old's desire to escape chore-related troubles! Using a technology stack of React, Node.js, Express, and MongoDB, I transformed this idea into a reality. The app offers unique features like chore assignment, difficulty levels, progress tracking, and family earnings management, turning daily chores into a game and fostering family collaboration.

Butterfly App

The Butterfly app serves as an accessible, educational resource for butterfly enthusiasts and curious minds alike. Users can delve into a comprehensive database, filled with various butterfly species, and gain insights into their physical characteristics, habitats, and behaviors. With high-quality images and a user-friendly interface, identifying and learning about butterflies has never been simpler. T his project was built using Python, Django, PostgreSQL, HTML, CSS, and leveraged the MaterializeCSS framework for an elegant user interface design.
